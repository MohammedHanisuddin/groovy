DYNAMIC PROGRAMMING

  • Dynamic programming is a recursive optimization procedure which means it is a procedure which optimizes step by step basis using information from the preceding steps
  • The two basic approaches for solving dynamic programming are:

-Forward recursion

-Backward recursion
